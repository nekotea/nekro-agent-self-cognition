# nekro-agent-self-cognition

NekroAgent 自我认知记忆插件。让 Bot 拥有自己的观点与偏好，记住参与过的讨论，了解群友的个人信息，并能主动判断是否加入群聊话题。

## 功能概览

- **话题记忆**：Bot 在对话中形成的观点、立场、参与过的讨论，存入向量数据库，按语义召回
- **角色画像**：Bot 对群友的了解（职业、爱好、宠物、设备、游戏、项目等），独立存储与召回
- **LLM 主动刷新**：Bot 发现上下文中的记忆与当前话题不匹配时，自行调用工具更新记忆槽，零自动嵌入开销
- **网络搜索**：对时效性或不熟悉的话题，Bot 可调用百度千帆搜索获取信息，整理成观点后入库，每日限额自动熔断
- **主动参与监控**：群聊有 ≥N 条未回复消息且超过阈值时间后，自动触发 LLM 判断是否参与讨论

## 工作流程

```
每次 LLM 响应前
  └─ 提示注入（零 API 调用）
       ├─ 话题记忆槽（最多 2 个）：上次刷新的 Bot 观点
       └─ 角色画像槽（最多 4 个）：上次刷新的群友画像

LLM 判断记忆是否匹配当前话题
  ├─ 不匹配 → 调用「刷新话题记忆」/ 「刷新角色画像」（语义搜索 Qdrant → 更新槽）
  ├─ 话题不熟悉 → 调用「搜索网络」获取信息 → 整理观点 → 「记录认知」入库
  └─ 形成新认知 → 调用「记录认知」（自动去重/更新）

后台任务（每 5 分钟）
  └─ 群聊 ≥2 条未回复 & 超过 15 分钟 → 触发 LLM 主动判断是否参与
```

## 安装

将 `plugin.py` 放入 NekroAgent 的插件目录（`plugins/workdir/`），文件名改为 `self_cognition.py`，在管理面板启用插件。

**依赖**：NekroAgent 需已配置 Qdrant 向量数据库和 Gemini Embedding 服务（主系统统一管理，插件直接复用）。

## 配置项

在管理面板 → 插件设置中配置：

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `MAX_TOPIC_SLOTS` | 2 | 同时注入的最大话题记忆槽数 |
| `MAX_PERSONA_SLOTS` | 4 | 同时注入的最大角色画像槽数 |
| `MAX_SLOT_CONTENT_CHARS` | 300 | 单槽注入内容最大字符数 |
| `SLOT_TTL` | 7200 | 槽过期时间（秒），闲置超时后自动失效 |
| `RECALL_TOP_K` | 4 | 每次刷新最多召回的记忆条数 |
| `RECALL_SCORE_THRESHOLD` | 0.72 | 召回相似度阈值（0-1） |
| `DEDUP_THRESHOLD` | 0.90 | 写入去重阈值，超过此值则更新而非新建 |
| `SEARCH_API_KEY` | 空 | 百度千帆 ai_search API Key，留空禁用搜索 |
| `SEARCH_DAILY_LIMIT` | 50 | 每日搜索次数上限，超出后当日自动禁用 |
| `SEARCH_MAX_RESULTS` | 5 | 每次搜索返回给 LLM 的最大结果条数 |
| `PROACTIVE_ENABLED` | true | 是否启用主动监控 |
| `PROACTIVE_TRIGGER_MINUTES` | 15 | 未回复讨论超过多少分钟后触发 |
| `PROACTIVE_MIN_MESSAGES` | 2 | 触发所需的最少未回复消息数 |
| `PROACTIVE_CHECK_INTERVAL` | 300 | 后台任务检查间隔（秒） |

### 百度搜索配置

`SEARCH_API_KEY` 填入百度千帆 `bce-v3/ALTAK-...` 格式的 Key（[申请地址](https://console.bce.baidu.com/qianfan/tools/toolsCenter)），留空则搜索功能不可用。每日用满 `SEARCH_DAILY_LIMIT` 次后次日自动恢复。

## LLM 可用工具（沙盒方法）

| 工具名 | 类型 | 触发时机 |
|--------|------|---------|
| `刷新话题记忆` | AGENT | 当前话题与注入的记忆不匹配，或涉及新话题 |
| `刷新角色画像` | AGENT | 对话涉及某群友但画像槽中无此人或内容过时 |
| `记录认知` | BEHAVIOR | 形成明确观点、或得知群友具体信息时 |
| `搜索网络` | AGENT | 遇到时效性强或不熟悉的话题时主动搜索 |

### 记录认知格式示例

**Bot 自身观点：**
- topic: `音乐偏好` / content: `我偏好电子音乐，lo-fi 很适合写代码时听`
- topic: `对 AI 伦理的看法` / content: `我认为 AI 辅助创作是工具，不会取代创作者的核心价值`

**群友认知：**
- topic: `群友[小明]的游戏爱好` / content: `小明最近在玩怪物猎人和 CSGO，我问了怪猎用什么武器，他说是弓`
- topic: `群友[阿强]的职业背景` / content: `阿强是后端工程师，主要写 Go，最近在做微服务迁移`

## 调试

插件在 `DEBUG` 日志级别输出所有关键调用：嵌入模型调用（耗时、维度）、Qdrant 搜索/写入（命中分数）、百度搜索 API（配额、耗时、结果数）、所有沙盒工具入参、槽状态与注入内容、主动监控每轮检查详情。
